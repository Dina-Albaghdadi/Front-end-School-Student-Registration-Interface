# Front-end-School-Student-Registration-Form
Project Overview
This project is a web-based interface designed for a Student Management System (CUBED SMS). Its primary purpose is to digitize student registration processes by providing an organized, user-friendly electronic form to collect and manage academic records.

Features
Comprehensive Data Entry: Over 20 input fields including ID tracking, personal details (name, sex, age, birth date), and parental information (occupation, employer).
Media Support: Includes a dedicated section for uploading and displaying student profile pictures.
System Operations: UI includes functional action buttons for system-level tasks: Add, Save, Delete, and Database access.
Interactive Elements: Utilizes HTML5 features such as dropdown selection boxes and date pickers for accurate data entry.

Technologies Used
HTML5: Used for the semantic structure of the registration form and page elements.
CSS3: Linked via an external stylesheet (Style.css) to provide a professional dark-green themed layout and responsive positioning.

Project Structure:
Index.html: The core structure of the registration form, 
Style.css: The external stylesheet governing the visual presentation.
